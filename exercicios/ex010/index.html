Como fiz isso

Primeiro, coloquei o comando <a href="O link vai aqui">Aqui vai o texto que representa o link</a>

Para o target tem quatro comandos

_blank - vai abrir o link em uma nova janela em branco.
_self - vai abrir o link na janela ou frame atual(padrão).
_top - vai desfazer todos os frames e abrir o destino navegador completo.
_parent - similar ao uso do _top em uma referência à janela mãe.

para o rel tem cinco comandos

next - indicar que o link é para a proxima parte do documento atual.
prev - indica que o link é para a parte anterior do documento atual.
author - indica que o link é para o site do autor doartigo atual.
external - indica que o link é para outro site que nãofaz parte do site atual.
nofollow - indica é um link para um site não endossado, como um link pago.

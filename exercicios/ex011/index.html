Imagem Dinâmica

Abrir em tamanhos diferentes mudam a Imagem
Para isso usei as seguintes tags
<picture> | comando crtl+espace selecionando source:media:type | <img>
Esse <source> possue três atributos
media = indica o tamanho maximo a ser considerado para carregar a imagem indicada no atributo srcset
srcset = vai configurar o nome da imagem que será carregada quando o tamanho indicado for atingido
type = indica o tamanho maximo a ser considerado para carregar a imagem indicada no atributo
